GrainScanAppMFN – Streamlit-додаток для аналізу зерна
=====================================================

Опис
-----
GrainScanAppMFN – це науково-практичний додаток для аналізу якості зерна.
Він дозволяє:
- Завантажувати зображення зерна (JPEG/PNG).
- Виявляти та рахувати зерна на зображенні.
- Розраховувати площу кожного зерна.
- Оцінювати рівномірність зерна (коефіцієнт варіації).
- Виконувати класифікацію зерен за площею (спрощений аналіз дефектів).
- Виявляти сторонні частинки (індикатори забруднення).
- Будувати гістограму розподілу площ зерен.
- Виводити зведену статистику (кількість, середня, мінімальна, максимальна площа).
- Автоматично порівнювати результати з умовними еталонними стандартами.
- Експортувати результати у CSV.
- Візуалізувати контури зерен (зелені рамки) та забруднень (червоні рамки).

Структура папки
---------------
GrainScanAppMFN/
├── GrainScanAppMFN.py           # Основний Streamlit-код
├── run_grainscanMFN.bat         # Файл для запуску програми (Windows)
├── requirements.txt             # Список необхідних бібліотек
├── test_images/                 # Папка для тестових зображень
└── README.txt                   # Інструкція для запуску

Вимоги
------
- Python 3.9–3.11
- Віртуальне середовище (venv або conda)
- Бібліотеки з requirements.txt:
  streamlit, opencv-python, numpy, pandas,
  scikit-image, matplotlib, scikit-learn

Алгоритм запуску
----------------
1. Встановіть Python (перевірка: `python --version`).
2. Створіть віртуальне середовище:
   `python -m venv venv`
3. Активуйте середовище:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. Встановіть залежності:
   `pip install -r requirements.txt`
5. Запустіть додаток:
   - Через .bat файл: подвійний клік на run_grainscanMFN.bat
   - Або вручну: `streamlit run GrainScanAppMFN.py`
6. Відкрийте браузер за адресою:
   http://localhost:8501
7. Завантажте тестове зображення з папки `test_images/` та перегляньте результати.

Примітки
--------
- Автоматичне порівняння з еталонними стандартами використовує умовні пороги
  (середня площа 100–500 пікселів, CV ≤ 0.30, кількість ≥ 20).
  Для наукових досліджень ці значення слід адаптувати відповідно до реальних ДСТУ/ISO норм.
- Якщо з’являється помилка `ModuleNotFoundError`, додайте відсутній модуль у requirements.txt та виконайте `pip install`.
- Для коректної роботи бажано використовувати чіткі зображення зерна на контрастному фоні.
- Результати можна експортувати у CSV для подальшої обробки.
